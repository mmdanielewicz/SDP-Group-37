## Running LLM locally
### Dependencies
- Ollama (install locally at https://ollama.com/download)
	- Restart computer after installation to use in terminal
- Llama 3.1:8b
	- Run: ```ollama pull llama3.1:8b```
- Ollama python module
	- Run: ```pip install ollama```
